# 好きなアーティスト曲ランキング

このWebページは、FRUITS ZIPPER・藤井風・ONE OK ROCK の曲ランキングを表示します。  
ランキングは色分けされ、スクロールに応じてフェードインする演出付きです。  
背景はCSSアニメーションによる流れるグラデーションで、ページ全体が華やかになります。

## 特徴
- ランキングは各アーティストごとに1位〜3位まで表示
- 色分けされたランキングで視覚的に分かりやすい
- アーティスト画像と簡単な紹介文付き
- 曲タイトルをクリックすると公式YouTube動画を別タブで再生
- スクロールでセクションがフェードインするアニメーション
- 流れるグラデーション背景で動きのあるデザイン

## 技術スタック
- HTML / CSS / JavaScript
- CSSアニメーション（背景グラデーション）
- IntersectionObserverによるスクロールフェードイン


## 注意事項
- 曲リンクは公式YouTube動画にリンクしています
